单行注释–，多行注释/……/

数据库->表->列->数据

查询所有数据库：show database；

查询当前数据库：select database（）；

创建: create database

查询当前所有表： show tables;

查询表结构：desc ()；

查询指定表的建表语句： show create ()

创建字段：create table

添加字段:alter table表名 ADD 字段名 类型（长度）

删除字段 ：alter table 表明 drop 字段名

修改表名：alter table 表名 rename to 新表名；

删除表：drop table 表名

添加数据

批量添加数据：insert into 表名（字段名1，字段名2….）values（值1，值2..），（值1，值2）

insert into 表名 values（值1，值2），（值 1，值2）

删除数据

查询数据

EOF通常使用两个小于号（<<）和一个标识符来定义，该标识符可以是任何有效的字符串，通常是大写字母E后跟一个唯一的字符串，以指示这是一个EOF标识符。然后，在EOF标识符上方和下方放置多行文本，这些文本将被视为一个整体字符串。

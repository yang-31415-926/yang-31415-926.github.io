sql

sql语句可以单行书写也可以多行书写，以分号结尾可使用空格，缩进增加可读性，sql对大小写不敏感

单行注释–，多行注释/……/

数据库->表->列->数据

查询所有数据库：show database；

查询当前数据库：select database（）；

创建: create database

查询当前所有表： show tables;

查询表结构：desc ()；

查询指定表的建表语句： show create ()

创建字段：create table

添加字段:alter table表名 ADD 字段名 类型（长度）

删除字段 ：alter table 表明 drop 字段名

修改表名：alter table 表名 rename to 新表名；

删除表：drop table 表名

添加数据

给指定字段添加数据:insert into表名（字段名1，字段名2）values （值1，值2）；

给全部字段添加数据：insert into 表名 values（值1，值2）

批量添加数据：insert into 表名（字段名1，字段名2….）values（值1，值2..），（值1，值2）

insert into 表名 values（值1，值2），（值 1，值2）

修改数据

update 表名 set 字段名1=值1，字段名2=值2，…where 条件；

删除数据

delete from 表名 where 条件

查询数据

swlect 字段名 from 表名 where 条件

分组查询

select 字段列表 from where条件 表名 group by 分组字段名 having条件

排序查询

swlect字段列表 from表名 other by 字段1 排序方式 字段2 排序方式（在字段1相同的情况下才会再去比较字段2）

排序方式：升序desv 降序 asc

分页查询

select 字段列表 from表名 LIMIT 起始索引，查询记录数

起始索引=（查询页码-1）*每页显示记录数

编写顺序 SELECT->FROM->WHERE->GROUP BY->HAVING->ORDER BY->LIMIT

执行顺序：FROM->WHERE->GROUP BY->HAVING->SELECT->ORDER BY->LIMIT

管理用户

查询用户：USE mysql；select* from user

创建用户：create user 用户名 @ 主机名 identified by 密码 ；

修改用户密码： alter user ”用户名 @ 主机名 identified with mysql_native_password BY 新密码 ；

删除用户：drop user 用户名 @ 主机名；
